## Installation
Clone the repository and run index.html in a web browser, or play [here](https://cdn.rawgit.com/16point7/one-offs/snakes/snakes/index.html).

## How to Play
* **left:** ArrowLeft
* **right:** ArrowRight
* **down:** ArrownDown
* **up:** ArrowUp
* **faster:** KeyF
* **slower:** KeyS
